博客综述——数据科学版块
2024-01-08

「合抱之木，生于毫末；九层之台，起于垒土；千里之行，始于足下」

​ ——《 老子・德经・第六十四章 》

​ 经过了一天的忙碌，我的个人博客终于搭建好了，这确实是想当令我感到欣喜的。之所以选择在互联网上单独搭建一个自己的博客，而不是选择一些现有的平台，如知乎，哔哩哔哩等，是因为相较于复杂的社交平台的环境，我更喜欢那种有一片属于自己的空间的感觉。

​ 本版块主要发布的内容的主题为数据科学，主要是我在学习过程中对于一些较为重要的知识的总结，包括但不限于以下内容：

  • Python 编程内容及数据分析的第三方库（numpy、pandas、matplotlib 等）
  • Machine Learning 的经典算法及其代码实现
  • Deep Learning 的经典模型算法及其底层代码实现
  • 基于 pytorch 框架的 Deep Learning 模型搭建
  • 计算机视觉方向主流算法及经典论文复现
  • 自然语言处理方向主流算法及经典论文复现

​ 这四部分基本上会涵盖整个机器学习和深度学习的核心内容。当然，博客的更新也将伴随我的学习生涯。至于我为什么选择自己去总结一些内容，有以下几点原因：

  1. 该领域有大量的复杂概念及数学推导，但互联网上已有的博客很多都是对于公式进行罗列和简单说明，有一些容易产生问题和疑惑的地方解释并不详细，导致数学推导部分相当的抽象，难于理解。
  2. 费曼学习法告诉我们，要想更加透彻地学会一部分内容，把它给别人讲会是最好的方法。我对此经历过亲身实践并深信不疑。
  3. 市面上有很多主流的人工智能方向的书籍，我将尽可能把我认为优秀的书籍的内容进行整合，力求每一部分的内容尽可能全面，详尽，有深度。

​ 你可以把我的博客当做一个资料库，更可以把它当做一本数据科学方向的讲义，因为我将在我的博客中把我所理解到的知识尽我所能分享给你。最后，请允许我介绍一下该板块博客的参考书籍：

《模式识别（第四版）》 张学工 汪小我编著

《DEEP LEARNING 深度学习》 [美] lan Goodfellow [加] Yoshua Bengio [加] Aaron Courville 著

《机器学习及其 Python 实践》 阚道宏编著

《Python 机器学习实战》 吕云翔等编著

《机器学习》 周志华著
